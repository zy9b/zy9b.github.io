如何用Adobe完成系列暑假作业(Window10及以上）

作者：得瑟的唐三藏

下载Adobe的破解软件

Adobe,一个你们很熟悉的厂商，Ps就是这个坑货的。
但不得不说，Adobe是真的好用！ 本教程无广告，无花费，只需要准备好一个足够大的C盘来喂Adobe全家桶的无底洞就行了。

1.下载GenP

GenP是一个外网的破解软件，它的原理是断掉Adobe的砧板服务，从而实现可以在下载正版的基础上正常访问和更新Adobe,毫不夸张地说，这个小东西是全球唯一一个可以支持Adobe自己更新的破解器。 我已经从外网下载好了GenP,点击我开始下载

2.下载CCStopper

CCStopper是Github上的一位大佬推出的程序，可以掐断Adobe的网络服务（不包括更新服务） Github下载太慢了，所以我又帮你下好了，点击我开始下载

Adobe下载

Adobe的传送门~~~ 安装时记得掐掉正版服务~~~

破解

  • 打开C:\Program Files (x86)\Common Files\Adobe\Adobe Desktop Common\AppsPanel（一般都在这个文件下）里面有三个文件和两个文件夹
  • 解压GenP 3.0后，找到GenP 3.0 Release\GenP 3.0 Release\Utilities\restore install buttons [beta]，里面有一个readme和两个文件
  • 用GenP的文件替换掉Adobe的三个文件（一个都不能留！）
  • 启动Adobe Creative Cloud
  • 解压CCStopper后，找到CCStopper\CCStopper_v1.3.0-pre.1\CCStopper,运行，输入2和1，随后退出
    • 至此，Adobe Creative Cloud破解完毕，你可以安装自己喜欢额软件了，破解结果： 安装软件点击安装即可~~~
